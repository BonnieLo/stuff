This is (or will be) a molecule viewer with molecular dynamics (done in the cloud) for minimization and simulation. You can import structures from a library or cut and paste a structure into a pop-up text window.

I'm not sure whether it should allow you to build structures in the GUI, the way NanoCAD did. I guess it would pretty lame if it didn't.

Drag with mouse to rotate.

Do a simulation run, which the user can view as an animation

Do energy minimization to find the minimal-energy arrangement of atoms in a molecule.

Think about parameters, e.g. step size, temperature, etc

If users sign in, they can download an MPEG of their simulation run.
